# Introdução ao MATLAB - Inteligência Artificial Aplicada (Automação)

Este repositório contém atividades e exercícios desenvolvidos utilizando o MATLAB, com foco no aprendizado de conceitos fundamentais de programação e análise numérica.

## Objetivo

O objetivo deste projeto é desenvolver habilidades básicas em MATLAB, incluindo:

- Manipulação de dados
- Operações matemáticas
- Estruturas de repetição e decisão
- Leitura e escrita de arquivos
- Visualização de dados
- Implementação de algoritmos simples

---

## Conteúdo do Repositório

O repositório inclui scripts que abordam diferentes tópicos, como:

### Manipulação de Dados
- Leitura de arquivos `.txt`
- Organização de dados em matrizes
- Normalização de valores

### Algoritmos de Aprendizado
- Implementação de modelos simples como:
  - Regra de Hebb
  - Perceptron

### Processamento Numérico
- Operações com vetores e matrizes
- Cálculos matemáticos aplicados

### Visualização de Dados
- Plotagem de gráficos
- Análise visual de resultados

---

## Estrutura do Projeto

Os arquivos estão organizados em scripts MATLAB (`.m`), onde cada um representa uma atividade ou experimento específico.

Exemplo:

- `script_Hebb.m` → Implementação da regra de Hebb  
- `script_perceptron.m` → Algoritmo de classificação  
- `Tabela_dados.txt` → Base de dados utilizada nos treinamentos  

---

## Como Executar

1. Abra o MATLAB
2. Clone o repositório:
```bash
git clone https://github.com/EricIkeda1/IntroducaoMatlab.git
```
3. Navegue até a pasta do projeto
4. Execute os scripts `.m` no MATLAB
